## 2. 処理フロー詳細

### 統合ダッシュボード（`GET entame/index.php`）
1. **必須条件**: ログイン済み（`Auth::requireLogin()`）
2. **集計**:
   - 映画: `UserMovieModel::countByStatus('watchlist'|'watched')`, `getTotalWatchedRuntime()`
   - ドラマ: `UserDramaModel::countByStatus('wanna_watch'|'watching'|'watched')`, `getTotalWatchedRuntime()`
   - アニメ: `UserWorkModel::getStatsByUser($userId)`
3. **画面**: `private/apps/Entame/Views/entame_dashboard.php` に渡してHTML描画
4. **例外**: 例外時はログ出力し、集計値は0埋めで表示を継続

### 映画追加（`POST movie/api/add.php`）
1. **必須条件**: ログイン済み（未ログインは 401）
2. **入力**: `tmdb_id`（必須）、`status`（watchlist/watched）、必要に応じて `tmdb_data`
3. **作品キャッシュ**:
   - TMDB設定があれば `TmdbApiClient` で詳細取得、`MovieModel::findOrCreateByTmdbId()` で `mv_movies` を upsert
4. **ユーザーリスト追加**:
   - `UserMovieModel::addMovie(movie_id, status)` で `mv_user_movies` を作成
5. **出力**: 成否JSON（UI側は toast とボタン状態更新）

### 映画：手動追加（仮登録）（`POST movie/api/add_manual.php`）
1. **入力**: `title`, `status`
2. **仮登録**: `mv_movies.tmdb_id=NULL` のプレースホルダーを作成（同名があれば再利用）
3. **ユーザーリスト追加**: `mv_user_movies` を作成
4. **補足**: 後から `movie/api/link_tmdb.php` でTMDB映画へ紐付け可能

### アニメ：ステータス変更（`POST anime/api/set_status.php`）
1. **必須条件**: ログイン済み
2. **入力**: `work_id`（AnnictWorkId）, `kind`（wanna_watch/watching/watched/no_select）
3. **キャッシュ補完**: `an_works` に存在しない場合は Annict API から取得し upsert
4. **ローカル保存**: `an_user_works` を upsert（※Annict本体には書き込まない）
5. **出力**: `status=success|error` のJSON

### おすすめ（映画/ドラマ）（`GET */api/recommendations.php?type=...`）
1. **必須条件**: ログイン済み（未ログインは 401）
2. **入力**: `type` = `personal` / `genre` / `trending`
3. **キャッシュ**:
   - TTL: 6時間（Controller定数 `REC_CACHE_TTL=21600`）
   - 保存先（ファイルキャッシュ）:
     - 映画: `private/apps/cache/rec/` 配下（typeごと、trendingは共通）
     - ドラマ: `private/apps/cache/drama_rec/` 配下（typeごと、trendingは共通）
4. **TMDB未設定**: `status=error, message=TMDB未設定` を返す

## 3. 外部連携 I/O（環境変数・認証）
### TMDB
- **環境変数**: `TMDB_API_KEY`
- **認証方式**: クエリ `api_key` で送信
- **主要クライアント**:
  - 映画: `App\\Movie\\Model\\TmdbApiClient`
  - ドラマ(TV): `App\\Drama\\Model\\TmdbTvApiClient`
- **共通パラメータ**: `language=ja-JP`, `include_adult=false`

### Annict OAuth（アニメ）
- **環境変数**: `ANNICT_CLIENT_ID`, `ANNICT_CLIENT_SECRET`, `ANNICT_REDIRECT_URI`
- **スコープ**: `read`（現状）
- **トークン保存先**: `an_annict_tokens`
- **コールバック**: `GET anime/oauth_callback.php`（code/error を受け取り、exchange→保存）

### アニメβ制限（任意）
- **環境変数**: `ANIME_BETA_ID_NAMES`（カンマ区切り `id_name`）
- **制限のかかるAPI例**:
  - `POST anime/api/set_status.php`（403 Forbidden）
  - `POST anime/api/revoke.php`（403 Forbidden）
