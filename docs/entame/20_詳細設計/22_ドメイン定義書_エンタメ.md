## 2. ステータス・区分値定義 (マジックナンバー)
- `mv_user_movies.status`:
  - `watchlist` = 見たい
  - `watched` = 見た
- `dr_user_series.status`:
  - `wanna_watch` = 見たい
  - `watching` = 見てる
  - `watched` = 見た
- `an_user_works.status`:
  - `wanna_watch` = 見たい
  - `watching` = 見てる
  - `watched` = 見た
  - `on_hold` = 保留
  - `stop_watching` = 視聴中止
