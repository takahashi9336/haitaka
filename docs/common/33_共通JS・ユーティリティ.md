# 共通JS・ユーティリティ

## 1. グローバルコア（必読）

| 項目 | 内容 |
|------|------|
| **ファイル** | [`www/assets/js/core.js`](../../www/assets/js/core.js) |
| **公開API** | `window.App`（二重読込はガード済み） |

### App の主要メソッド

| メソッド | 説明 |
|----------|------|
| `App.toast(message, duration?)` | 画面上部中央の汎用トースト（`#app-toast` を生成）。`flash_toast.php` のセッションベース表示とは別 |
| `App.post(url, data)` | `fetch` で JSON POST。`response.ok` でなければ `{ status: 'error', message }` を返す。ネットワークエラー時はユーザ向け文言に正規化 |
| `App.calculateRemaining(dueDateStr)` | 期日までの日数（切り上げ） |
| `App.initScrollPersist()` | `DOMContentLoaded` で自動実行。`data-scroll-persist="キー"` 付き要素の `scrollTop` を `sessionStorage` に保持 |
| `App.buildBackUrl(basePath, paramKeys)` / `App.saveListParams(basePath)` | 一覧のクエリを `sessionStorage` に保存し、戻るリンク用に復元 |
| `App.goBack(fallback)` | 同一オリジンの `referrer` があれば `history.back`、なければ `fallback`（既定 `/hinata/`）へ |
| `App.initSidebar()` | プレースホルダ（処理なし）。サイドバー制御は `sidebar.php` 側 |
