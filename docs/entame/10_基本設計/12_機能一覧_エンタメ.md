# エンタメ（アニメ・ドラマ・映画） 機能一覧

## 1. 画面一覧
| 画面名 (論理名) | ファイルパス | 概要 |
| :--- | :--- | :--- |
| エンタメ統合ダッシュボード | `www/entame/index.php` → `private/apps/Entame/Views/entame_dashboard.php` | 映画/ドラマ/アニメの統計を合算・メディア別に表示し、各機能へ遷移する |
| 映画ダッシュボード | `www/movie/index.php` → `private/apps/Movie/Views/movie_dashboard.php` | 映画のスタッツ、検索、ガチャ、友人の視聴、レコメンド、グラフを表示 |
| 映画一覧（見たい/見た） | `www/movie/list.php` → `private/apps/Movie/Views/movie_list.php` | 映画リストの表示・並び替え・フィルタ・追加導線（TMDB） |
| 映画詳細 | `www/movie/detail.php` → `private/apps/Movie/Views/movie_detail.php` | 映画の詳細表示、視聴日/評価/メモ、タグ、TMDB連携など |
| 映画検索ページ | `www/movie/search.php` → `private/apps/Movie/Views/movie_search.php` | 検索結果の表示（API呼び出しのUI） |
| 映画一括登録 | `www/movie/import.php` → `private/apps/Movie/Views/movie_import.php` | 一括追加（プレビュー→確定） |
| 映画一括編集 | `www/movie/bulk_edit.php` → `private/apps/Movie/Views/movie_bulk_edit.php` | 一括更新/削除 |
| ドラマダッシュボード | `www/drama/index.php` → `private/apps/Drama/Views/drama_dashboard.php` | ドラマのスタッツ、検索、ガチャ、友人の視聴、レコメンド、グラフを表示 |
| ドラマ一覧 | `www/drama/list.php` → `private/apps/Drama/Views/drama_list.php` | ドラマのリスト表示（grid/list）・絞り込み・追加導線（TMDB） |
| ドラマ詳細 | `www/drama/detail.php` → `private/apps/Drama/Views/drama_detail.php` | ドラマの詳細表示、ステータス/評価/メモ/タグ/進捗等 |
| ドラマ検索ページ | `www/drama/search.php` → `private/apps/Drama/Views/drama_search.php` | 検索結果の表示（API呼び出しのUI） |
| ドラマ一括登録 | `www/drama/import.php` → `private/apps/Drama/Views/drama_import.php` | 一括追加（プレビュー→確定） |
| アニメダッシュボード | `www/anime/index.php` → `private/apps/Anime/Views/anime_dashboard.php` | アニメのスタッツ、検索（Annict）、ガチャ、今期、友人の視聴、グラフを表示 |
| アニメ一覧 | `www/anime/list.php` → `private/apps/Anime/Views/anime_list.php` | ステータス別（見たい/見てる/見た）の一覧 |
| アニメ詳細 | `www/anime/detail.php` → `private/apps/Anime/Views/anime_detail.php` | Annict/ローカルキャッシュを併用して詳細を表示 |
| 今期アニメ一覧 | `www/anime/current_season.php` → `private/apps/Anime/Views/anime_current_season.php` | Annictから今期作品を取得し一覧表示 |
| アニメ一括登録 | `www/anime/import.php` → `private/apps/Anime/Views/anime_import.php` | Annict OAuth 連携状態の案内と一括取り込み導線 |
| アニメ検証/導線 | `www/anime/verify.php` → `private/apps/Anime/Views/anime_verify.php` | 導入/検証向けの画面（存在確認） |

## 2. 機能・アクション一覧
| 機能名 | 種類 (画面/API/Batch) | 概要 |
| :--- | :--- | :--- |
| エンタメ統合サマリ表示 | 画面 | 映画/ドラマ/アニメの件数・視聴時間を集計して表示 |
| 映画：検索（TMDB） | API | `/movie/api/search.php` でTMDB検索し、追加ボタンを提供 |
| 映画：追加 | API | `/movie/api/add.php` でTMDB映画をリストに追加 |
| 映画：手動追加（仮登録） | API | `/movie/api/add_manual.php` でタイトルのみの仮登録を作成 |
| 映画：更新（ステータス/評価/メモ/視聴日） | API | `/movie/api/update.php` でユーザー映画エントリを更新 |
| 映画：タグ更新 | API | `/movie/api/update_tags.php` でタグ（JSON配列）を更新 |
| 映画：削除 | API | `/movie/api/remove.php` でリストから削除 |
| 映画：ガチャ | API | `/movie/api/gacha.php` 見たいリストからランダム抽選 |
| 映画：おすすめ | API | `/movie/api/recommendations.php?type=...`（personal/genre/trending） |
| 映画：TMDB紐付け | API | `/movie/api/link_tmdb.php` 仮登録映画にTMDB情報を紐付け |
| 映画：一括登録 | API | `/movie/api/bulk_add.php` まとめて追加 |
| 映画：一括更新 | API | `/movie/api/bulk_update.php` まとめて更新/削除 |
| 映画：配信情報バッチ | Batch | `www/movie/batch_providers.php`（配信情報更新用） |
| ドラマ：検索（TMDB） | API | `/drama/api/search.php` |
| ドラマ：追加 | API | `/drama/api/add.php` |
| ドラマ：更新（ステータス/評価/メモ等） | API | `/drama/api/update.php` |
| ドラマ：タグ更新 | API | `/drama/api/update_tags.php` |
| ドラマ：削除 | API | `/drama/api/remove.php` |
| ドラマ：ガチャ | API | `/drama/api/gacha.php` |
| ドラマ：おすすめ | API | `/drama/api/recommendations.php?type=...` |
| ドラマ：TMDB詳細（配信情報等） | API | `/drama/api/tmdb_detail.php` |
| ドラマ：一括登録 | API | `/drama/api/bulk_add.php` |
| アニメ：検索（Annict） | API | `/anime/api/search.php` |
| アニメ：ステータス更新 | API | `/anime/api/set_status.php`（ローカルDBのみ） |
| アニメ：ガチャ | API | `/anime/api/gacha.php` |
| アニメ：今期取得 | API | `/anime/api/current_season.php` |
| アニメ：OAuth コールバック | API/画面 | `/anime/oauth_callback.php`（認可コード受け取り） |
| アニメ：連携解除 | API | `/anime/api/revoke.php` |
| 外部連携：TMDB API | 外部 | `TmdbApiClient`（映画）/ `TmdbTvApiClient`（ドラマ） |
| 外部連携：Annict OAuth/API | 外部 | `AnnictOAuthService`（OAuth）/ `AnnictApiClient`（API） |

## 3. 関連テーブル一覧
| テーブル物理名 | テーブル論理名 | 役割（CRUDの種別など） |
| :--- | :--- | :--- |
| `mv_movies` | 映画マスタ（TMDBキャッシュ/仮登録） | 参照中心＋TMDB更新（U） |
| `mv_user_movies` | ユーザー映画リスト | 追加/更新/削除（CRUD） |
| `dr_series` | ドラマ（TVシリーズ）マスタ（TMDBキャッシュ） | 参照中心＋TMDB更新（U） |
| `dr_user_series` | ユーザードラマリスト | 追加/更新/削除（CRUD） |
| `an_annict_tokens` | Annict OAuthトークン | 追加/更新/削除（ユーザー単位） |
| `an_works` | Annict 作品キャッシュ | 検索・補完用（Upsert） |
| `an_user_works` | ユーザーアニメリスト | 追加/更新/削除（CRUD） |

| `sys_role_apps` | ロール×アプリ | `movie` / `hinata_movie` に entame配下を紐付け（参照） |
| `sys_users` | ユーザー | 各ユーザーリストのFK参照元（参照） |
