# live_trip 変更履歴・設計決定録 (ADR)

## 2026-04-29 設計書初版（docs 体系整備）

* **変更内容**: `docs/live_trip/` に基本設計・詳細設計・本 ADR を追加。実装は `private/apps/LiveTrip` / `www/live_trip` を根拠とした。
* **背景・理由**: AI・開発者が同一リファレンスを共有するため。旧 `docs/99_old/遠征管理_DB設計_ER図.md` の思想を最新スキーマに合わせて再記述。
* **見送った代替案**: コントローラと 1:1 の巨大 I-O 表（全メソッドのパラメータ列挙）はメンテコストが高いため、一覧表＋代表フローに留めた。

## 2026-04-29 遠征タイトル導入とイベント後追い紐付け

* **変更内容**: `lt_trip_plans.title` を導入し、遠征作成時の必須入力をイベント選択からタイトル入力へ変更。イベント紐付けは任意（0件可）とし、作成後に編集画面から追加可能にした。
* **背景・理由**: 「イベントを先に作らないと遠征管理を開始できない」UX課題を解消し、遠征を独立プロジェクトとして管理するため。
* **見送った代替案**: `lt_trip_plan_events` 側に擬似タイトルイベント（プレースホルダ）を登録する案は、データ責務が曖昧になるため不採用。
