## 4. 実装（画面ごとの統一作業）

### 4.1 画面共通で揃えるチェックリスト

- **高さ/余白**: `h-14` + `px-4`
- **背景/枠/影**: `bg-white` + `border-b` + `shadow-sm`
- **sticky/z-index**: `sticky top-0` +（z-index値を統一）
- **Left構造**:
  - `#mobileMenuBtn`（SPのみ）
  - 戻る（必要画面のみ、`fa-chevron-left`）
  - アイコン（`w-8 h-8 rounded-lg`）
  - タイトル（`truncate`）
- **Right構造**:
  - 個数制約は無し（複数配置可）
  - ただしボタンサイズ/角丸/タイポは統一する
